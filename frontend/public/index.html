Présentation et utilisation

Mondial Relais est une application web qui facilite le lien entre Mondial Relay et l'utilisateur, d'où le nom de notre entreprise.

La problematique que nous avons rencontré est la modernisation des apis de Mondial Relay, qui utilisent un langage de formatage obsolète (le XML). Notre but étant de moderniser tout ça en permettant à l'utilisateur de faire des requêtes à l'api de Mondial Relay dans des formats modernes (JSON, formulaire html), afin d'obtenir des réponses plus simples d'utilisation et intuitives pour l'utilisateur.

Notre application possède plusieurs fonctionnalités utilisant l'API de mondial relay :

  • Créer une étiquette d'expédition Mondial Relay pour vos colis : Pour ce faire, il vous suffit de remplir le formulaire, puis de cliquer sur "Télécharger l'étiquette". Votre étiquette se trouvera ensuite, au format PDF, dans le dossier réceptionnant les téléchargements par défaut sur votre ordinateur.

  • Recherche de point relais : Pour ce faire, rendez vous sur la partie "Recherche point relais" du site, puis remplissez le formulaire qui vous sera présenté. Une fois le bouton de recherche cliqué, une carte apparaîtra avec des points représentant les Mondial Relay. Cliquez sur un point pour afficher les informations du Mondial Relay sélectionné.

  • Nous possédons également une boîte gmail si vous avez des question ou des remarques concernant notre application. Pour ce faire, accédez à la partie "Contact" puis remplissez le formulaire.

  • Une partie de création de compte et connexion est actuellement en cours de développement, merci de votre patience.

Aspect technique

Cette application se base sur l'API officielle de Mondial Relay utilisant des requêtes et réponses au format XML pour créer l'étiquette. En amont, nous avons développé un serveur permettant d'envoyer des requêtes au format JSON, qui seront par la suite convertis au format XML pour faire la requête à l'api de Mondial Relay. Par la suite, le serveur reçoit une réponse XML qui sera convertie en JSON pour faciliter l'exploitation de la réponse.
